# UMDB Frontend

React frontend for the Universal Media Database.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will run on http://localhost:5173

## Features

- **Browse Movies**: View all movies in your collection with search and filters
- **Movie Details**: View comprehensive information about each movie
- **Add Movie**: Manual entry form for adding movies from physical media
- **Search External**: Search and import from TMDB and IMDB

## Tech Stack

- React 18
- TypeScript
- React Router for navigation
- Axios for API calls
- Tailwind CSS for styling
- Vite for fast development

## Build

```bash
npm run build
```

The production build will be in the `dist/` directory.
